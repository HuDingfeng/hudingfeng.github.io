git使用

git 绑定本地仓库 上传git服务器

1.获取仓库地址 例如:https://www.hjkljhjhjhjh.com/dafff

2.打开终端 cd 文件夹目录

3.git init

4.git remote add origin master https://www.hjkljhjhjhjh.com/dafff(换成你自己的仓库地址)

5.git add . //注意add和.之间是有空格的 表示把所有文件添加到本地缓存

6.git commit -m "你的更新或者提交内容描述"

7.git push -u origin master//将你刚commit的文件全部提交git服务器
