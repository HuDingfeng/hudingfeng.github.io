前言
偶然接触到hexo，花了一些时间搭建了下，也遇到了一些坑 ，记录下来分享。主要是在mac oS下。
下载node.js&&注册github

下载node.js
（略）

注册github账号

创建一个仓库名为”xxx.xxx.github.io”(xxx是你的用户名)。

配置Hexo

安装Hexo

npm install hexo-cli -g

创建个文件夹 例如”blog”

cd 文件夹路径

hexo init
然后你的hexo就初始化在了这个文件夹中， 然后我们接着；

hexo generate

hexo server

现在我们可以在本地服务器查看本地编辑的blog了。http://0.0.0.0:4000/ 到此为止我们的本地配置已经结束。
